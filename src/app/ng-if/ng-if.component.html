ng-if.component: *ngIf

*ngIf="condition": Content is rendered when condition is true

Content to render when condition is true.

---------------------------------------------------------------------------------

*ngIf="condition2; else elseBlock2": dùng với ng-template

content là child của div
Content to render when condition is false.

---------------------------------------------------------------------------------

*ngIf="condition; then thenBlock else elseBlock": dùng với ng-template

content là child của div
Content to render when condition is true.
Content to render when condition is false.

-----------------------------------------------------------------------------------
